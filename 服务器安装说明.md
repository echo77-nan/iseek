# 服务器安装说明

## 问题：ModuleNotFoundError: No module named 'pymysql'

这是因为服务器上还没有安装 Python 依赖包。

## 解决方案

### 方法1：使用 pip3 安装（推荐）

在服务器上执行以下命令：

```bash
cd /home/echo.ln/iseek/backend
pip3 install -r requirements.txt
```

如果权限不足，使用 `--user` 参数：

```bash
pip3 install --user -r requirements.txt
```

### 方法2：使用虚拟环境（最佳实践）

```bash
cd /home/echo.ln/iseek/backend

# 创建虚拟环境
python3 -m venv venv

# 激活虚拟环境
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt

# 启动服务（在虚拟环境中）
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### 方法3：使用安装脚本

```bash
cd /home/echo.ln/iseek/backend
bash install_dependencies.sh
```

## 验证安装

安装完成后，验证依赖是否安装成功：

```bash
python3 -c "import pymysql; import fastapi; import dashscope; print('所有依赖已安装')"
```

## 启动服务

安装完成后，启动服务：

```bash
cd /home/echo.ln/iseek/backend
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

## 常见问题

### 1. 如果 pip3 命令不存在

```bash
# 安装 pip3
sudo yum install python3-pip
# 或
sudo apt-get install python3-pip
```

### 2. 如果安装速度慢

使用国内镜像源：

```bash
pip3 install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 3. 检查 Python 版本

确保 Python 版本 >= 3.8：

```bash
python3 --version
```
